Este programa está formado por cuatro clases:

- La clase Main contiene el método main.

- La clase Animal incluye el número de crías y tiene tres métodos:
  - El método que pide sus datos.
  - El método que visualiza sus datos.
  - El metodo que alimenta al animal.

- La clase Mamifero hereda de la anterior y además contiene el número de huesos y el número de extremidades.

- La clase Persona hereda de la anterior y además contiene la profesión.
