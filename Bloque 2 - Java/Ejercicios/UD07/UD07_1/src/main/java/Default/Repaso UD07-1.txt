CLASES
------
Arrays: Mismas características que en C.
 - Excepción: el tamaño puede ser definido a partir de una variable.
 - Las matrices no tienen por qué ser cuadradas.
 - La propiedad length nos da el dato del tamaño.
 - En Java se declara y se reserva espacio en dos operaciones distintas (si bien pueden hacerse al mismo tiempo).
 - Al igual que en C, los arrays son "punteros".


Si el array es de objetos hay que:
- Crear el array.
- Invocar al constructor de la clase de esos objetos una vez por
cada "celda".


Cadenas: "Tipo" String.
- Son objetos, de tamaño no definido.
- Cuando se cambia el contenido lo que se cambia realmente es la referencia.
- La clase String proporciona un buen número de métodos.


Clases "envoltorio": Integer, Float, Double, etc.
- Los métodos "parse" me sirven para hacer conversiones de cadena a cada uno de esos tipos.
